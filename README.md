# Cascade

小さな習慣が滝のように積み重なり、一日の流れをつくる自己管理 PWA。

スマートフォンのホームスクリーンに追加して、SNS を開く前に最初に開く場所として設計。

## 機能

- **Morning Gate** — 毎朝1回の起動儀式。今日の言葉・意図設定・連続日数の積層ビジュアル
- **習慣スタック** — 8習慣のタップ完了、達成率リング、今週のヒートマップ
- **タスク** — ローカルタスクの追加・完了・削除
- **夜の記録** — 達成習慣のサマリー・一言日記・満足度・明日の最重要タスク

## 設計思想

- Atomic Habits（小さく・簡単に・満足感）
- Digital Minimalism（通知なし、選んだものだけ）
- ストリーク > 完璧主義

## 技術

純粋な HTML / CSS / JavaScript（フレームワーク不使用）。データは端末内の localStorage に保存され、外部に送信されない。Service Worker でオフライン動作。

## 使い方

1. スマートフォンのブラウザでこの URL を開く
2. 「ホームスクリーンに追加」
3. アプリとして起動 → 毎朝 Morning Gate が表示される
